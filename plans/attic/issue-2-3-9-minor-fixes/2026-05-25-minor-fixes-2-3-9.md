## Task 1: eidos#2 — Null-safe comparisons (TDD)

**Files:**
- Modify: `persistence-memory/src/test/java/io/casehub/eidos/memory/InMemoryAgentRegistryTest.java`

The slot filter (`query.slot().equals(d.slot())`) is idiomatically unsafe — `String.equals(null)` doesn't NPE but the intent is unclear. The JPA entity enforces `nullable=false` but in-memory has no such guard.

- [ ] **Step 1: Add comment above `substring(0, 16)` in `sha256()`**

Replace lines 248–249 in `ClaudeMarkdownRenderer.java`:

```java
    private static String sha256(final String input) {
        try {
            final var digest = MessageDigest.getInstance("SHA-256");
            final byte[] hash = digest.digest(input.getBytes(StandardCharsets.UTF_8));
            // 16 hex chars = 64 bits. Collision probability (1/2^64) is negligible for
            // cache-invalidation fingerprinting — hashes are compared for equality only,
            // not used for cryptographic trust.
            return HexFormat.of().formatHex(hash).substring(0, 16);
        } catch (final NoSuchAlgorithmException e) {
            throw new IllegalStateException("SHA-256 not available", e);
        }
    }
```

```java
    /**
     * Renders {@code desc} + {@code ctx} via a capturing ChatModel and returns the
     * full prompt string that was sent to the LLM (YAML + preamble).
     */
    private String renderAndCaptureYaml(final AgentDescriptor desc, final AgentPromptContext ctx) {
        final String[] captured = new String[1];
        final ChatModel capturingLlm = new ChatModel() {
            @Override
            public ChatResponse doChat(final ChatRequest request) {
                captured[0] = request.messages().stream()
                        .filter(m -> m instanceof UserMessage)
                        .map(m -> ((UserMessage) m).singleText())
                        .reduce("", (a, b) -> a + b);
                return ChatResponse.builder().aiMessage(AiMessage.from("rendered")).build();
            }
        };
        new ClaudeMarkdownRenderer(capturingLlm, new CdiVocabularyRegistry())
                .render(desc, ctx);
        return captured[0];
    }
```
